# FLAMES Game in Python

This is a fun Python program that determines the relationship between two people using the FLAMES game.

## Features
- Takes two names as input
- Removes common characters
- Calculates relationship using FLAMES logic
- Displays the final relationship result

## Technologies Used
- Python
- String manipulation
- Lists and loops

## How to Run

1. Run the program:
   python flames.py

2. Enter two names when prompted.

## Example

Input:
Player 1: Meena  
Player 2: Arun  

Output:
Relationship status: Friends
